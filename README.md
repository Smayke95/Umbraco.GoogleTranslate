# Umbraco.GoogleTranslate
Plugin for Umbraco CMS which integrates Google Translate.

## Configuration
Add configuration to appsettings.json

```
"GoogleTranslate": {
  "ApiKey": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
}
```
